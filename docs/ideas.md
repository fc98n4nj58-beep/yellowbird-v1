# Ideas

## Content Excellence / Product Differentiation Pass

Future phase after core launch readiness reaches 100%.

Purpose:

After core launch readiness reaches 100%, shift from stability/QA mode into a deeper quality and differentiation pass. The goal is to make Yellow Bird feel meaningfully better than a basic worksheet generator or generic resource catalog.

This phase should not distract from current launch-readiness work. It begins only after core launch readiness is complete or explicitly chosen.

### Worksheet Quality Review

* Improve prompt quality
* Reduce mechanical/repetitive feel
* Make tasks feel teacher-created, not machine-produced
* Review age appropriateness and cognitive load

### Pedagogy And Scaffolding

* Add better progression within worksheets
* Include easier-to-harder sequencing where appropriate
* Add "show your thinking" moments
* Support multiple strategies and representations

### Differentiation

* Easier / standard / challenge variants
* Support intervention, practice, enrichment
* Add extension questions where useful
* Consider ELL-friendly wording and accessibility

### Visual And Print Presentation

* Improve worksheet layout hierarchy
* Better use of whitespace
* More polished visual models
* Cleaner answer areas
* Stronger preview thumbnails/display

### Teacher-Facing Value

* Add "When to use this"
* Add "What students practise"
* Add prep notes where useful
* Add curriculum alignment notes
* Add estimated time / grouping suggestions

### Catalog/Display Improvements

* Better resource detail pages
* Stronger previews
* Better collections
* Related resources
* Strand/topic landing pages
* Teacher-friendly browsing pathways

### Content Voice And Brand

* Calm, warm, professional wording
* Encouraging student-facing lines
* Consistent Yellow Bird tone
* Avoid flashy/cluttered design
* Maintain print-first trust

### Product Differentiation

* Make resources feel curated, not dumped
* Build worksheet families/collections
* Add purposeful variety
* Improve resource discovery around teacher intent
* Eventually connect worksheets into lessons/units when launch core is stable
